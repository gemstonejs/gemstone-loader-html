About
-----

This is the Webpack Loader for HTML files in the
[Gemstone JavaScript Technology Stack](http://gemstonejs.com).
It is primarily based on [PostHTML](https://github.com/posthtml/posthtml) and [VueJS](https://vuejs.org/).
